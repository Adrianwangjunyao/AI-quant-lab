# 技术指标交互计算器 —— 产品设计文档

## 1. 产品概述

### 1.1 产品定位
一个纯前端的交互式技术分析工具，让用户通过**选择股票 → 调节参数 → 即时重绘**的闭环流程，直观理解 RSI、MACD、布林带、ATR、KDJ 这 5 个指标在不同参数下的表现差异。

### 1.2 目标用户
- 学习技术分析的新手（快速理解各指标参数的含义）
- 需要调试/优化策略的股票投资者（小范围试参数）
- 教学演示场景（课堂上展示参数变化对指标的影响）

### 1.3 核心差异化
| 对比维度 | 传统股票软件（同花顺/通达信） | 本工具 |
|----------|------------------------------|--------|
| 参数调节 | 需进入设置菜单，修改后全局生效 | **侧边栏滑动条即时调节**，所见即所得 |
| 对比能力 | 不支持同一指标不同参数对比 | 可通过截图/多开对比 |
| 数据源 | 需要接券商数据 | **支持 CSV 上传** + 预设数据 |
| 部署 | 需安装客户端 | 单 HTML 文件，双击即用 |

## 2. 产品架构

### 2.1 四层架构

```
┌─────────────────────────────────────────────────────┐
│                   数据输入层                          │
│  CSV上传  │  预设股票下拉  │  日期范围选择            │
└────────────────────┬────────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────────┐
│                   计算引擎层                          │
│  RSI  │  MACD  │  布林带  │  ATR  │  KDJ            │
│           纯前端 JavaScript 计算                     │
└────────────────────┬────────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────────┐
│                   可视化层                            │
│  价格走势（对数/线性） │  指标子图  │  信号标注        │
└────────────────────┬────────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────────┐
│                   交互反馈层                          │
│  参数即时重绘  │  信号摘要  │  导出 PNG               │
└─────────────────────────────────────────────────────┘
```

### 2.2 关键技术选型

| 模块 | 技术方案 | 选择理由 |
|------|----------|----------|
| 图表绘制 | **Chart.js** | 体积小(70KB)、交互性好(缩放/悬停)、Canvas 渲染性能佳 |
| 布局 | 原生 HTML + CSS Grid/Flex | 无需框架，单文件即可运行 |
| 数据解析 | 原生 FileReader API | 解析 CSV，无额外依赖 |
| 指标计算 | 纯 JavaScript 函数 | 可直接从 Jupyter Notebook 的 Python 代码翻译 |
| 日期处理 | 原生 Date API | 无需 moment.js，需求简单 |

## 3. 功能规格

### 3.1 数据管理

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 预设股票 | P0 | 内置 3~5 只 A 股示例数据（含兆易创新） |
| CSV 上传 | P0 | 用户上传自己的 CSV 文件（自动检测列名） |
| 日期范围 | P1 | 通过滑块或日期选择器筛选展示时间窗口 |
| 数据表格 | P2 | 可展开查看原始数据 |

### 3.2 指标计算与参数

#### RSI（相对强弱指标）
| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| 周期 N | 滑动条 | 14 | 3~30 | 计算周期越短越敏感 |
| 超买阈值 | 滑动条 | 70 | 60~90 | 超过此值标记超买 |
| 超卖阈值 | 滑动条 | 30 | 10~40 | 低于此值标记超卖 |

#### MACD（指数平滑异同移动平均线）
| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| 快线(EMA12) | 数字输入 | 12 | 5~30 | 短期 EMA |
| 慢线(EMA26) | 数字输入 | 26 | 10~60 | 长期 EMA |
| 信号线(DEA) | 数字输入 | 9 | 3~20 | DIF 的 EMA 周期 |

#### 布林带（Bollinger Bands）
| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| 周期 | 滑动条 | 20 | 5~50 | 移动平均周期 |
| 标准差倍数 | 滑动条 | 2 | 1.0~4.0 | 倍数越大通道越宽 |

#### ATR（平均真实波幅）
| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| 周期 | 滑动条 | 14 | 5~30 | Wilder 平滑周期 |

#### KDJ（随机指标）
| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| RSV 周期 | 数字输入 | 9 | 3~30 | N 日高低范围 |
| K 平滑 | 数字输入 | 3 | 2~10 | K 值平滑系数 |
| D 平滑 | 数字输入 | 3 | 2~10 | D 值平滑系数 |

### 3.3 图表功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 多指标同屏 | P0 | 上下分割子图，价格 + 指标联动 |
| 对数/线性切换 | P0 | 价格图坐标切换 |
| 指标显示开关 | P0 | 每个指标可单独显示/隐藏 |
| 悬停数值 | P1 | 鼠标悬停显示对应日期的指标值 |
| 缩放/平移 | P1 | 基于 Chart.js 的交互缩放 |
| 金叉死叉标注 | P1 | 在图表上自动标注交叉点 |
| 信号共振热力图 | P2 | 类似 Notebook 中的信号矩阵 |

### 3.4 交互流程

```
选择股票 ──→ 加载数据 ──→ 查看默认指标图
   │                           │
   │                    ┌──────┴──────┐
   │                    │             │
   │                调节参数      切换显示
   │                    │             │
   └──→ 重新选择 ──→ 即时重绘 ←────┘
                           │
                      ┌────┴────┐
                      │         │
                  查看信号    导出图片
```

## 4. 界面设计

### 4.1 布局结构

```
┌──────────────────────────────────────────────────────────┐
│   HEADER: Logo + 股票选择器 + 加载按钮                    │
├──────────┬───────────────────────────────────────────────┤
│          │                                               │
│  左侧    │  [RSI] [MACD] [布林带] [ATR] [KDJ]  ← 标签页 │
│  参数面板 ├───────────────────────────────────────────────┤
│          │                                               │
│  RSI ◄───┤  ┌ 价格走势（对数坐标）──────────────────┐    │
│  MACD    │  │ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━    │    │
│  布林带  │  └──────────────────────────────────────┘    │
│  ATR     │  ┌ 指标曲线 ────────────────────────────┐    │
│  KDJ     │  │ ~~~~~~~~ RSI / MACD / KDJ ~~~~~~~~~  │    │
│          │  └──────────────────────────────────────┘    │
│  [重新    │  [快速统计: 收盘价 / RSI / MACD / %B / ATR] │
│   计算]   │                                               │
├──────────┴───────────────────────────────────────────────┤
│   FOOTER: 信号摘要 + 导出 + 切换全部指标                   │
└──────────────────────────────────────────────────────────┘
```

> **关键区别**：每个指标拥有独立的标签页（Tab），切换标签页时左侧面板高亮对应指标的参数卡，右侧图表区域切换为对应的指标图。
> 每个指标图由上下两个子图构成：上图为价格走势 + 该指标叠加（如布林带通道），下图为纯指标曲线（如 RSI/MACD/KDJ）。

## 5. 数据流设计

### 5.1 CSV 输入格式

与现有数据保持一致，自动检测以下列名：

```
必要列: trade_date, open, high, low, close, vol
可选列: amount, pre_close, change, pct_chg
```

### 5.2 状态管理

```
状态对象：
{
  stockData: RawData[]       // 原始数据
  params: IndicatorParams   // 5 个指标的当前参数
  visibleIndicators: Set    // 当前显示的指标
  chartInstance: Chart      // Chart.js 实例
  dateRange: [start, end]   // 日期筛选范围
  scaleType: 'linear'|'log' // 坐标类型
}
```

### 5.3 重绘流程

```
用户滑动滑块
  → requestAnimationFrame 节流（16ms）
  → 更新 params 对象
  → 调用 computeAllIndicators()
  → 返回计算结果对象 {rsi, macd, bb, atr, kdj}
  → chartInstance.data.datasets 更新
  → chartInstance.update('none') 无动画更新
```

## 6. 文件结构

```
交易指标交互计算器.html     # 主文件（单 HTML，所有代码内嵌）
│
├── HTML: 布局结构
├── CSS: 全部样式
└── JS:
    ├── dataParser.js       # CSV 解析、数据校验
    ├── indicators.js       # 5 个指标的计算函数
    ├── chartRenderer.js    # Chart.js 渲染与更新
    ├── signalAnalyzer.js   # 金叉/死叉检测、信号汇总
    └── app.js              # 状态管理、事件绑定
```

> 实际交付为**单 HTML 文件**，上述 JS 模块在 `<script>` 中分区组织。

## 9. 局限与优化方向

**已知局限：**
- 全部数据加载到内存，超过 5000 行可能需要优化（数据抽稀）
- 不支持实时行情（仅历史数据分析）
- 不包含选股功能（仅指标展示）

**后续可扩展：**
- 增加更多指标（OBV、CCI、WR%等）
- 增加多股同列对比功能
- 增加策略回测（根据信号买卖的历史收益率）
- 增加 PWA 支持（可离线安装）
